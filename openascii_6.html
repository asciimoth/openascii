Open ASCII Art Archive This project (WIP) aims to collect ASCII art that can be used in derivative works (games, CLI tools, etc.) without any copyright issues and provide it in an easy-to-use form.
ASCII Art

ASCII art is a design technique that composes text characters to create visual shapes and illustrations.

Original ASCII art used only characters from the ASCII standard, but nowadays this term is used for text art in general.

There is a lot of awesome ASCII art created over the decades-long history of this phenomenon, but unfortunately a huge part of it has uncertain authorship and an undefined license, which makes its use in derivative works questionable (although everyone uses it anyway).

Licenses & Authorship This archive contains only art with known authorship and explicitly defined licenses (mostly permissive or copyleft ones).
Always check the license for each specific art you want to use to find out the terms of use.
Respect your ASCII artists by always leaving creds (one way or another) to original artwork even if licence don't requires it.
Contribution If you are interested in contributing your artwork to this archive or would like your artwork removed, open an issue or PR for this project on GitHub or contact me.
Keymaps
  • hjkl or ←↓↑→ - navigation
  • JK - right panel scrolling
  • gG - page up/down
  • d - download selected art
  • s - go to the selected art's source
  • S - open the selected art's source in a new tab
  • o - open the same selection in a new tab
