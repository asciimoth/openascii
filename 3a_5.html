Animated ASCII Art 3a is a text file format designed for storing ASCII art. It supports 8-bit ANSI and RGB colors as well as animations, while preserving text layout, unlike plain text with ANSI codes, which have historically been used for storing ASCII art.
All art in this archive is provided in 3a format.
Tooling You can use the aaa tool to play 3a animations in the terminal, convert them to other formats (PNG, GIF, WebP, MP4, asciicast, etc.), edit them, and more. For developers, there are parser libraries available: rs3a, py3a, go3a.
